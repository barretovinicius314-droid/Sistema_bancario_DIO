# Sistema_bancario_DIO
💰 Sistema Bancário em Python

Este projeto é um sistema bancário simples desenvolvido em Python. Ele permite realizar operações básicas como depósito, saque e consulta de saldo (extrato).

🚀 Funcionalidades
 Depositar valores na conta
 Realizar saques com regras de limite
 Consultar extrato e saldo atual
 Controle de número máximo de saques diários
📌 Regras do Sistema
💵 Depósito
Apenas valores positivos são aceitos.
O valor é adicionado ao saldo.
A operação é registrada no extrato.
